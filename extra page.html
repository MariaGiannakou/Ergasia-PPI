Αρχική Πρότζεκτ Βιογραφικό Επικοινωνία Οδηγός GIT


Εντολές που χρησιμοποιήθηκαν στο GIT.

  1. git config --global user.email : Ο χρήστης βάζει το όνομά του ώστε να αναγνωρίζει το τερματικό ποιος κάνει τις αλλαγές.

  2. git config --global user.name : Ο χρήστης βάζει το email του ώστε να αναγνωρίζει το τερματικό ποιος κάνει τις αλλαγές.

  3. git clone : Κλωνοποιεί ένα repository σε νέο κατάλογo ώστε να κατέβουν τα αρχεία στον υπολογιστή.

  4. git init : Δημιουργεί ένα κενό repository Git ή επανεγκαταστεί ένα ήδη υπάρχον για να αποκτήσει έλεγχο ο χρήστης.

  5. git add "name of file" : Προσθέτει το περιεχόμενο του αρχείου.

  6. git add . : Προσθέτει όλα τα αρχεία.

  7. git mv : Μετακινεί ή μετονομάζει ένα αρχείο, έναν κατάλογο ή ένα σύμβολο συμβόλων.

  8. git rm : Καταργεί αρχειά από το δέντρο εργασίας και από το ευρετήριο.

  9. git merge : Συμμετοχή σε δύο ή περισσότερες ενέργειες ανάπτυξης μαζί.

  10. git status : Εμφάνιση της κατάστασης tree εργασίας.

  11. git pull : Κατεβάζει και ενοποιεί με ένα άλλο repository ή ένα τοπικό branch ώστε όλοι οι χρήστες να έχουν τις τελευταίες ενημερώσεις.

  12. git push : Ενημερώνει απομακρυσμένα αναφορές μαζί με συναφή αντικείμενα ώστε όλοι οι χρήστες να έχουν τις τελευταίες ενημερώσεις.

  13. git fetch : Κατεβάζει αντικείμενα και αναφορές από άλλο repository και συνήθως χρησιμοποιείται σε collaborator.

  14. mkdir "name of a file" : Δημιουργεί έναν φάκελο στον τοπικό δίσκο.

  15. cd "name of a file" : Μπαίνει στον φάκελο που αναγράφεται.

  16. touch : Δημιουργεί ένα αρχείο μέσα σε έναν φάκελο.
