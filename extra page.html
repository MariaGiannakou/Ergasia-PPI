Εντολές που χρησιμοποιήθηκαν στο GIT.

  1. git config --global user.email : Ο χρήστης βαζει τα στοιχεία ώστε να αναγνωρίζονται
  2. git config --global user.name : Ο χρήστης βαζει τα στοιχεία ώστε να αναγνωρίζονται
  3. git clone : Κλωνοποιήστε έναν repository σε νέο κατάλογo
  4. git init : Δημιουργήστε ένα κενό repository Git ή επανεγκαταστήστε ένα υπάρχον
  5. git add : Προσθήκη περιεχομένου αρχείου
  6. git add . : Προσθήκη όλων των περιεχομένων αρχείου
  7. git mv : Μετακινήστε ή μετονομάστε ένα αρχείο, έναν κατάλογο ή ένα σύμβολο συμβόλων
  8. git rm : Κατάργηση αρχείων από το δέντρο εργασίας και από το ευρετήριο
  9. git merge : Συμμετοχή σε δύο ή περισσότερες ιστορίες ανάπτυξης μαζί
  10. git status : Εμφάνιση της κατάστασης δέντρου εργασίας
  11. git pull : Φέρτε και ενοποιήστε με ένα άλλο repository ή ένα τοπικό branch
  12. git push : Ενημέρωση απομακρυσμένων αναφορών μαζί με συναφή αντικείμενα
  13. git fetch : Κατεβάστε αντικείμενα και αναφορές από άλλο repository
